# Project Name

Marketing and Ordering Website with built-in CMS for New World Park Solutions.

## Tech Stack
- **Frontend:** React, Next.js, Tailwind CSS
- **Database:** PostgreSQL
- **Other tools:** ESLint, Prettier

## Installation

Step-by-step instructions to get the project running locally.

```bash
# Clone the repository
git clone https://github.com/Zinazero/nwps-website.git

# Go to project folder
cd nwps-website

# Install dependencies
npm install

# Start the development server
npm start
